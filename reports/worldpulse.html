WorldPulse

Real-time global intelligence dashboard: Bloomberg Terminal meets CNN War Room with 4 switchable editorial personalities

TypeScript Next.js 15 Leaflet Zustand Recharts shadcn/ui Bun SSE Vercel

The Challenge

No single dashboard aggregates news, conflict data, financial markets, and live video in one place.

Bloomberg Terminal costs $25K/year and focuses on finance. CNN covers news but not markets. ACLED tracks conflicts but has no news context. Individually, each source is incomplete. What is needed is a unified command center where geopolitical events, their market impact, and live footage converge in real time.

WorldPulse pulls from 6+ data sources (GNews, RSS feeds, ACLED, CoinGecko, Alpha Vantage, YouTube), proxies everything through Next.js API routes with in-memory TTL caching, and renders it all in a dark-themed window-manager layout. Four editorial "personalities" (War Room, Analyst, Global Citizen, Futurist) re-rank the same data with different category weights, letting you switch perspective without changing sources.

Why It Matters

  • Multi-source intelligence fusion: news, conflicts, finance, videos, and trends aggregated into one view. Market correlations link financial moves to geopolitical events.
  • 4 editorial personalities: War Room prioritizes conflict and military news. Analyst weights economy and data. Global Citizen highlights humanitarian impact. Futurist focuses on emerging tech. Same data, completely different reading experience.
  • Real-time updates: SSE streaming from /api/stream with automatic polling fallback. Finance data refreshes every 30s, news every 60s, conflicts every 5 minutes.
  • Window manager layout: draggable, resizable panels inspired by Bloomberg Terminal. Each panel (map, news, conflicts, finance, video, trends) can be rearranged. Layout persists across sessions.
  • Country intelligence: deep-dive reports per country with power index, threat level, sanctions tracker, and relationship matrix between nations.

Key Insights

Graceful Degradation by Design

Every data source can fail independently without breaking the dashboard. Missing API keys trigger fallback to RSS feeds or mock data. The cache layer ensures stale data is served while a source is down. SSE failures degrade to polling. The system health panel shows exactly which sources are active, stale, or errored.

Personality-Driven Curation

The same 50+ articles are re-ranked by multiplying each article's category against a personality's weight vector (7 categories, each weighted 1-10). Switching from War Room to Futurist completely reshuffles the feed without refetching data. The accent color, icon, and tone of the entire UI shift with each personality.

Conflict-Market Correlations

The correlation engine links conflict events to financial asset movements. When the Ukraine conflict escalates, it highlights Wheat and Natural Gas price spikes. When Middle East tensions rise, Oil futures get flagged. This cross-domain intelligence is what makes WorldPulse more than a news reader.

Window Manager

The layout uses a custom WindowManager component where each panel (map, news, conflicts, finance, video, trends, geopolitics, countries) is independently positionable. Layout state persists via useLayoutPersist hook to localStorage, so your preferred arrangement survives page refreshes.

7 Active Conflicts Tracked

Static conflict definitions (Ukraine, Gaza, Sudan, Myanmar, Ethiopia, DRC, Syria) are enriched in real-time with ACLED event data. Each conflict has bounds for map zooming, status tracking (active/escalating/de-escalating/ceasefire/frozen), party lists, and a timeline of events.
