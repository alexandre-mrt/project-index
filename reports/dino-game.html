Dino Game

Chrome dino runner clone — single-file HTML game

A faithful recreation of the Chrome offline dinosaur game, built entirely in a single HTML file. The player controls a pixel-art dino that jumps over procedurally spawned cacti on an infinite scrolling ground. The game features gravity-based physics, progressive difficulty that increases obstacle speed with your score, AABB collision detection, and persistent high scores via localStorage.

Vanilla JS HTML Canvas CSS

The dino is drawn entirely with fillRect calls — no images or sprites. Its legs even animate differently depending on whether it is grounded or airborne, giving it a surprisingly lively feel for pure rectangles.
