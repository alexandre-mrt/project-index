Personal Website

Developer portfolio with an immersive 3D Lord of the Rings easter egg

Next.js 16 React 19 TypeScript Three.js GSAP Tailwind CSS v4

The Challenge

Standing out as a developer requires more than a resume.

In a field where everyone lists the same frameworks and buzzwords, a personal website needs to demonstrate craft, not just describe it.

A portfolio site is the first thing recruiters, collaborators, and hackathon judges look at. It needs to communicate identity, technical depth, and taste within seconds. The challenge is building something that is both functional (shows experience, projects, contact) and memorable (reflects personality and skill level beyond the content itself).

Why It Matters

  • Professional branding: a single page that consolidates identity, experience at EPFL, BSA presidency, hackathon wins, and blockchain projects into one cohesive narrative.
  • Technical showcase: the site itself is a portfolio piece. Using Three.js, GSAP scroll-driven animations, and React Three Fiber demonstrates frontend mastery beyond standard React apps.
  • Hidden depth: the /moria easter egg page reveals a full 3D journey through the Mines of Moria, showing the kind of creative engineering that a flat project list never could.

How It Works

Main Portfolio Page

A clean, sidebar-driven single-page layout. The left panel is fixed on desktop with name, title, navigation, and social links. The right panel scrolls through sections: Hero, About, Experience, Projects, Activity, and Contact. The sidebar uses IntersectionObserver to highlight the active section as the user scrolls.

The Moria Easter Egg

Navigating to /moria reveals a password gate styled as the Doors of Durin. Typing "mellon" (the Elvish word for friend, as in Tolkien's Fellowship of the Ring) unlocks a fully immersive 3D experience.

The 3D scene is a scroll-driven journey through Moria, powered by React Three Fiber and GSAP ScrollTrigger. A 1000vh scroll container maps to camera movement along a CatmullRom spline through six chapters:

Camera path through the mine: Dead Marshes -> Doors of Durin -> Tunnels z=85 z=25 z=-30 to -210 | v Bridge of <- Dwarrowdelf Hall <- S-curve between Khazad-dum (pillars, stairs) pillar columns z=-560 z=-230 to -535

Post-processing (Bloom + Vignette) via @react-three/postprocessing creates cinematic lighting. Each environment (swamp, door arch, tunnel walls, pillar hall, staircase, bridge) is a separate component loaded with Suspense for progressive rendering.

Key Insights

Scroll-Driven 3D Without a Game Engine

The Moria journey maps browser scroll position to a CatmullRom camera spline using GSAP ScrollTrigger. A 1000vh tall container gives fine-grained control. Camera movement is smoothed with a lerp factor (0.05) applied per frame, preventing jitter. This approach avoids the complexity of a full game loop while producing a cinematic result.

Progressive 3D Loading

The /moria route uses Next.js dynamic() with ssr: false to skip server-side rendering of the Three.js canvas. Each environment (door, tunnels, hall, vista) is wrapped in Suspense with a null fallback, so the scene loads incrementally as the user scrolls into new areas.

Sidebar Navigation with IntersectionObserver

The desktop sidebar highlights the current section using a native IntersectionObserver with a -20% 0px -60% 0px root margin. This asymmetric margin means a section becomes "active" when it enters the top 20% of the viewport, matching natural reading position rather than geometric center.

Design System

A consistent indigo accent (#818cf8) threads through the entire site: gradient text in the hero, nav indicator bars, hover states on project cards, skill pill borders, and the elvish door decorations. The dark background (#0a0a0b) with zinc text tones creates a developer-appropriate aesthetic without being harsh.

Easter Egg as Portfolio Piece

The /moria page is not just a gimmick. It demonstrates proficiency with WebGL, procedural geometry, scroll-based interaction design, and post-processing. The password gate ("mellon") ensures it rewards curiosity without cluttering the main portfolio experience.
