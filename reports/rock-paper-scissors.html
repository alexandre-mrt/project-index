Rock Paper Scissors

Pierre Papier Ciseaux — single-file HTML game

A clean, emoji-driven Rock Paper Scissors game in a single HTML file. The player picks rock, paper, or scissors against a random CPU opponent in a dark-themed arena UI. It tracks running scores for both sides, color-codes results (green for win, red for loss, yellow for draw) with matching border highlights on the pick cards, and includes a reset button to start fresh.

Vanilla JS HTML CSS

The entire game logic fits in a single WINS lookup object — { rock: 'scissors', paper: 'rock', scissors: 'paper' } — making win detection a one-liner comparison instead of a chain of if/else statements.
