Road Trips Website

Immersive National Geographic-inspired showcase of 15 spiritual and cultural road trips with interactive MapLibre maps and day-by-day itineraries

TypeScript Next.js 15 MapLibre GL JS Tailwind CSS 4 Biome Bun Static Export Vercel

The Challenge

Travel blogs are either walls of text or shallow photo galleries. Neither format conveys the day-by-day rhythm of a road trip.

Planning a 15-day spiritual road trip through Ethiopia or Iran requires understanding distances, driving conditions, scenic ratings, accommodation types, budget breakdowns, and practical info (visas, road conditions, packing lists). No single travel site combines all of this with interactive maps and magazine-level visual design.

This website presents 15 handcrafted road trips as immersive editorial content. Each trip has a hero section, an interactive MapLibre map showing the route with day markers, a vertical day timeline, detailed activity cards with scenic ratings, budget estimates, and practical travel info. The design takes inspiration from National Geographic magazine layouts: full-bleed images, rich typography, and data-dense cards that reward exploration.

Why It Matters

  • 15 trips across 12 countries and 7 regions: Morocco, Ethiopia, Iran, Turkey, Japan, South India, Mongolia, Peru-Bolivia, Romania, Madagascar, Oman, Ladakh, Guatemala, Senegal, Tajikistan-Kyrgyzstan.
  • Deeply typed data model: every trip, day, activity, accommodation, and budget is a strict TypeScript interface with readonly properties. No runtime surprises.
  • Each trip has a unique color palette: 15 distinct palettes (primary, secondary, accent, bg, text) ensure every trip page feels like a different magazine issue.
  • Static export: the entire site compiles to static HTML at build time via Next.js static export. Zero runtime server, perfect Lighthouse scores, instant page loads.
  • Interactive maps without API keys: MapLibre GL JS with free CartoDB Voyager tiles. No Mapbox key, no usage limits, no billing surprises.

Key Insights

One Color Palette Per Trip

Each of the 15 trips has a unique 5-color palette (primary, secondary, accent, bg, text). When you navigate from Morocco (terracotta and sand) to Iceland (volcanic black and glacier blue), the entire page transforms. This is managed through TRIP_PALETTES in colors.ts, injected as CSS custom properties per trip page.

Rich Data Model

The Trip type contains nested readonly structures: each Day has activities (typed: spiritual, cultural, nature, food, adventure, historical), accommodation (with type and price range), road conditions (highway, mountain, coastal, desert, village), and a 1-5 scenic rating. This depth means every trip detail page is data-rich without any free-form text ambiguity.

Zero Runtime Dependencies

The site is statically exported at build time. All 15 trip pages are pre-rendered as HTML. There are no API calls, no database, no server. MapLibre uses free CartoDB Voyager tiles (no API key needed). The result is a site that loads instantly, works offline after first visit, and costs nothing to host on Vercel's free tier.

Scroll-Driven Animations

The ScrollReveal component wraps elements with an Intersection Observer that triggers fade-in animations on scroll. The useCountUp hook animates stat numbers (15 journeys, 12 countries, 200+ days, 15,000+ km) from zero to their final value when they enter the viewport. Subtle but effective for the magazine feel.
