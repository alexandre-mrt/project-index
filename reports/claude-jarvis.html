Claude Jarvis

A markdown-driven personal OS that unifies calendar, tasks, network, news, hobbies, and decisions into a single conversational interface.

Markdown Claude Code Google Calendar MCP Telegram MTProto Config-as-Code

Personal OS · Active since March 2026

No backend, no database, no deployment. Just markdown files and natural language.

The Challenge

Personal productivity is fragmented across dozens of disconnected apps. Google Calendar for events, Todoist for tasks, a CRM for contacts, RSS for news, Duolingo for languages, Notes for decisions. None of them talk to each other. None of them know your context.

Core problem: No single tool coordinates your calendar, tasks, relationships, learning goals, and information diet into a unified intelligence layer. You spend more time managing tools than doing the work.

Every morning starts with context-switching between 5+ apps just to figure out what matters today. Relationships go cold because no system tracks who you haven't spoken to. Decisions are made and forgotten without a trace.

Why It Matters

  • Context-aware prioritization — one system that knows your calendar, deadlines, relationships, and goals can surface what actually matters each morning
  • Relationship maintenance — automatic flagging of contacts you haven't reached out to in 30+ days prevents network decay
  • Live intelligence — curated news topics searched daily, with deep-dive triggers for breakthroughs relevant to your work
  • Zero-friction interface — say "good morning" and get a complete briefing; say "add task" and it is tracked. Natural language, no UI to learn
  • Decision memory — every choice logged with context and reasoning, reviewable monthly to learn from patterns

Conversational control plane: Every module is a markdown file. Every action is a natural language command. "Add task", "log decision", "prep for call with X", "who should I reach out to" — Jarvis reads, writes, and searches the right files automatically.

News Configuration

Each morning, Jarvis web-searches these configurable topic areas:

  • Blockchain Development — EIPs, Solidity tooling, L2 updates, exploits, hackathons
  • Fundraising & Opportunities — recent raises, grant programs, token launches
  • AI Agents & Robotics on Blockchain — credible agent frameworks, autonomous protocols
  • ZK Proofs, TEEs & Identity — research papers, Mysten/Sui updates, arxiv preprints

Deep-dive rule: If any search reveals a significant breakthrough (protocol upgrade, major exploit, groundbreaking paper), Jarvis automatically runs additional searches and provides a detailed explanation of what happened, why it matters, and how it connects to current work.

Key Insights

Config-as-code personal OS. Every aspect of life management is a markdown file: editable, versionable, portable. No database migrations, no API rate limits, no subscription fees. Adding a new life domain is just creating a new .md file.

No traditional backend needed. Claude Code acts as the runtime. MCP provides calendar integration. Web search provides live intelligence. Telegram provides delivery. The entire "backend" is a folder of markdown files and a language model that knows how to read them.

Natural language as the only interface. The golden rule: "Talk naturally. If you're unsure of a command, just describe what you want." There are no buttons, no forms, no dashboards. The command reference exists for discoverability, but free-form language always works.

Relationship maintenance as a system. The network module treats contacts like living documents: who they are, what they need, how to help them, interaction history. The 30-day silence detector prevents relationships from going cold by accident.
