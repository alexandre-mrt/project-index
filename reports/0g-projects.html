0G Projects

7-project ecosystem for the 0G Network covering decentralized storage, AI agent tooling, and data marketplaces. Built for the $88.88M 0G ecosystem grant program.

TypeScript 0G Storage AI Agents Solidity 0G SDK Hono

The Challenge

0G Network is a high-performance decentralized AI and storage chain, but the ecosystem lacks developer tools and reference applications. Builders need SDKs, gateways, and example apps to adopt the storage and compute primitives. Without these, the network stays empty.

Gap: No "Pinata for 0G", no agent framework, no block explorer, no data marketplace. Every primitive needs tooling from scratch.

Why It Matters

This is not a single project but a full ecosystem play across three layers: storage infrastructure, AI agent primitives, and end-user tools. Each project fills a specific gap in the 0G developer experience, and together they form a coherent stack.

750+ tests across 6 projects. Each sub-project ships independently with its own CI, test suite, and documentation.

Tech Stack

  • Runtime: Bun
  • Language: TypeScript (strict mode across all projects)
  • API framework: Hono (lightweight, Cloudflare Workers compatible)
  • Storage: 0G SDK (@0gfoundation/0g-ts-sdk) + ethers v6
  • Database: Bun SQLite / Drizzle ORM
  • Smart Contracts: Solidity ^0.8.19, OpenZeppelin v5 (ERC-7857 INFT, marketplaces)
  • Frontend: Server-rendered HTML (Hono) + React/Vite (zerostorage dashboard)
  • CI: GitHub Actions on all repos

Key Patterns

Agent Kit API

import { quickAgent } from '0g-agent-kit';

const agent = quickAgent({
  name: 'TradingBot',
  inferenceEndpoint: 'https://api.openai.com',
  inferenceApiKey: process.env.OPENAI_API_KEY,
});

const result = await agent.run('Analyze ETH/USDC');

Storage Gateway SDK

import { ZeroStore } from 'zerostorage';

const store = new ZeroStore({ apiKey: 'zs_xxx' });
const { rootHash } = await store.upload(file);
const data = await store.download(rootHash);

Smart Contracts

ERC-7857 (INFT): Agent Kit mints AI agents as Intelligent NFTs with encrypted metadata, secure transfer, usage authorization, and cloning. AgentBazaar adds a 2.5% fee marketplace. DataVault uses a 5% fee data marketplace contract with subscription support.

Revenue Models

  • zerostorage: Freemium SaaS (1GB free, $9-199/mo tiers)
  • 0g-agent-kit: Open-source core + managed hosting ($29/mo per agent) + premium templates ($9-49)
  • agentbazaar: 2.5% marketplace fee on agent sales
  • datavault: 5% marketplace fee on dataset sales

Key Insights

Ecosystem strategy: Rather than building one product, this targets three complementary layers (storage, agents, tools) to become essential infrastructure for the 0G developer community. Each project is independently useful but gains network effects from the others.

  • Grant-aligned: Developer tooling and AI agents are 0G's top priorities for their $88.88M ecosystem program
  • ERC-7857 bet: Early adoption of the INFT standard positions the agent-kit and marketplace as reference implementations
  • Consistent architecture: All projects share Bun + Hono + TypeScript, reducing context switching and enabling code reuse
  • Test coverage: 750+ tests total across the ecosystem, each project ships with CI
