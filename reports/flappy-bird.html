Flappy Bird

Classic flappy bird clone — single-file HTML game

A polished Flappy Bird clone packed into a single HTML file. The bird flaps through an endless series of randomly generated pipe gaps against a gradient sky with drifting clouds. It features a three-state game loop (ready, playing, dead), smooth bird rotation driven by vertical velocity, pipe highlight shaders for a 3D effect, and localStorage-backed high scores.

Vanilla JS HTML Canvas CSS

The bird is drawn with Canvas arc and ellipse primitives — body, animated wing, eye, and beak — and its rotation is computed directly from vertical velocity (bird.vy * 3, clamped between -30 and 90 degrees), so it naturally tilts up on flap and nose-dives when falling.
