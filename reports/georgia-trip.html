Georgia Trip

Interactive road trip companion for a Georgia (Caucasus) adventure: Leaflet map with animated route, accordion itinerary, highlights gallery, and printable cheatsheet

HTML CSS Vanilla JS Leaflet.js PWA Service Worker Inter Font

The Challenge

Planning a road trip through Georgia requires a companion that works offline in mountain valleys with no signal.

Google Maps can plan a route, but it cannot show you a curated itinerary with cultural context, food recommendations, and photo previews for each stop. Travel guides are static PDFs. What is needed is an interactive, offline-capable web app that serves as map, itinerary, and travel companion in one.

This single-page application packs an entire Georgia road trip into one HTML file. A Leaflet map with animated route playback shows the journey day by day. An accordion-style itinerary presents each day's activities with side-panel images that change on hover. A highlights gallery showcases the best stops. A history section covers Georgia from ancient Colchis to modern independence. A printable cheatsheet view reformats everything for paper. And a service worker makes it all work offline.

Why It Matters

  • Single HTML file architecture: everything (CSS, JS, data) is embedded in one 1,880-line HTML file. No build tools, no bundler, no dependencies beyond Leaflet CDN.
  • Offline-first via PWA: a service worker (sw.js) caches the page, Leaflet tiles, and Wikipedia images. Once loaded, the site works entirely offline in remote Georgian valleys.
  • Animated route playback: a slider control with play/pause animates a car icon along the route. Day dots below the slider let you jump to any day. The map panel updates with stop details, photos, and activity tags.
  • Dual-view design: toggle between the full interactive site and a printable cheatsheet view. The cheatsheet strips all visuals and reformats the itinerary for A4 printing with Google Maps links for each day.
  • Georgian wine-and-gold palette: wine (#722F37) and gold (#D4A843) on cream (#F5F0E8) backgrounds. The color scheme reflects Georgian culture: wine country, gold-domed churches, warm stone buildings.

Key Insights

Zero Build, Maximum Portability

The entire app is a single HTML file. No build step, no bundler, no package.json. You can email it, open it from a USB drive, or host it on any static server. The only external dependency is Leaflet from CDN, and even that gets cached by the service worker for offline use.

Accordion Itinerary with Side Images

Each day card in the itinerary starts collapsed, showing just the day number, title, and a preview of stops. Clicking expands it to reveal the full activity list. Each activity row has an icon (adventure, culture, food, drive, worship) and hovering over a row changes the side panel image to show that specific location. This hover-to-preview pattern keeps the page compact while providing visual richness.

Animated Route Playback

The route slider is not just a progress bar. It drives a car icon smoothly along the actual polyline path on the map. The side panel updates in real time with the current stop's photo, title, description, and activity tags. Clicking play auto-advances the slider with interpolated animation between waypoints. Day dots below the slider provide quick jumps.

PWA Offline Strategy

The service worker uses a cache-first strategy with a versioned cache name. On first load, it pre-caches the HTML page, Leaflet library, fonts, and all Wikimedia images referenced in the trip data. Subsequent visits work entirely offline. The inline web app manifest enables "Add to Home Screen" on mobile, making it feel like a native app in the Georgian mountains.

Print-Ready Cheatsheet

The view toggle button switches between the full interactive site and a clean cheatsheet view. The cheatsheet strips all maps, images, and interactivity, reformatting the itinerary as a compact day-by-day list with Google Maps links. CSS @media print rules hide the nav, hero, and footer, producing a clean A4 document.
