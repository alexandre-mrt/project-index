DataViz Perfume

Interactive perfume exploration and visualization for COM-480 Data Visualization at EPFL

JavaScript D3.js HTML/CSS Data Visualization EPFL COM-480

The Challenge

Perfume is an invisible, multi-dimensional product that is nearly impossible to compare or discover without physically smelling it.

With thousands of fragrances on the market, each described by abstract note pyramids (top, heart, base), scent families, brands, and subjective reviews, choosing a perfume is an overwhelming experience. Traditional e-commerce filters (price, brand) miss the most important dimension: how a perfume actually smells relative to others.

This COM-480 Data Visualization course project at EPFL builds an interactive web visualization that lets users explore the perfume landscape through multiple coordinated views: scent similarity maps, note composition breakdowns, brand comparisons, and recommendation paths based on fragrance profiles.

Why It Matters

  • Multi-dimensional exploration: perfume data has hierarchical notes (top/heart/base), categorical families (floral, woody, oriental), continuous ratings, and temporal trends that need different visualization techniques to reveal.
  • Coordinated views: brushing and linking between visualizations (selecting a scent family in one view highlights related perfumes in all others) enables discovery patterns that no single chart can show.
  • Sensory data challenge: translating olfactory properties into visual encodings (color for scent family, size for intensity, position for similarity) is a genuine information design problem.
  • Academic context: part of EPFL's COM-480 course, requiring both rigorous data analysis and effective storytelling through visualization.

Key Insights

Encoding the Invisible

The core design challenge is mapping olfactory properties to visual channels. Scent families map naturally to color hues (warm tones for oriental, cool for fresh). Note intensity maps to size or opacity. Similarity between perfumes maps to spatial proximity in 2D projections. The key insight is that perfume notes are hierarchical (top fades first, base lingers), which calls for layered or stacked visual encodings.

Coordinated Multiple Views

The visualization uses D3's brush and link pattern to connect all views. Selecting a region in the similarity map filters the note composition chart and the brand explorer simultaneously. This enables discovery of non-obvious patterns, like a cluster of affordable perfumes that share the same rare base note as a luxury brand.

Perfume Recommendation

The recommendation view computes similarity between perfumes using Jaccard distance on their note sets and cosine similarity on their accord vectors. Given a selected perfume, it displays the top-N most similar fragrances as a linked graph, allowing users to "walk" through the scent space from one perfume to related ones.
