Project Index

This site. Single-file HTML catalog of all personal projects, with filters, search, and linked per-project reports. Self-maintained metadata for 50+ projects.

HTML CSS (inline) Vanilla JS Vercel

The Challenge

Too many projects, no browsable overview.

The portfolio spans blockchain, AI, web, travel sites, tools, and research across dozens of repos. README files are scattered, GitHub lists lack context, and "show me what you've built" is hard to answer without a canonical view.

Project Index is a single index.html file (no framework) that catalogs every project with metadata: description, stack, tags, features, date, repo location, optional live URL, optional GitHub link, and a linked HTML report. Filters by category, full-text search, auto-grouping into Blockchain / AI / Web / Travel / Tools / Games.

Why It Matters

  • Zero framework overhead: a single HTML file, no build step, instant Vercel deploy.
  • Self-describing metadata: every card ships with stack, tags, features, and a deep-dive report link — not just a name.
  • Deep reports on demand: clicking "Report" opens a standalone HTML page with ASCII architecture diagrams, challenge framing, and implementation notes.
  • Living catalog: updated whenever new projects land; currently 50+ projects across 7 categories.
